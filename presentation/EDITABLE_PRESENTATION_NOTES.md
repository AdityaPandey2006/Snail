# Group 16 Snail Presentation Notes

This folder contains an editable source file for the generated PDF:

- `generate_group16_presentation.py`

The script already includes slide content mapped to the presentation rubric:

- Objective and motivation
- Challenges and how they were solved
- Novelty
- Demo and analysis
- Future scope
- Individual contribution

If you want to change wording, edit the `SLIDES` list near the top of the script and run:

```powershell
python presentation/generate_group16_presentation.py
```

Output file:

- `presentation/Snail_Group16_Presentation.pdf`
